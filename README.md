# Digital Identity Management Application for Government Services in Poland

## Overview
This project is a prototype digital identity system that allows users to securely access multiple services using a single identity.

## Features
- User authentication (login + OTP)
- Digital identity profile (PESEL, personal data)
- Dashboard for accessing services
- Health services (medical records simulation)
- Police services (complaint & summons check)
- Pension system (account overview)

## Technologies Used
- Frontend: React
- Backend: Node.js (Express)
- Database: MongoDB
- Authentication: JWT + OTP simulation

## Live Demo
https://e-identity-pl.emergent.host

## Purpose
This project demonstrates a centralized identity system to improve accessibility, security, and efficiency in digital services.
